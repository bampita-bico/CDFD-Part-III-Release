under CC BY 4.0.

a different software license, such as AGPL-3.0-or-later, without changing the

In practical terms:

- cite this archive for the Part III Adaptive Flux Limitation biology-and-medicine manuscripts;
- cite Part I for CDFL, the public CDFD framework notation;
- cite Part II where Life Number or origin-of-life boundary logic is reused;
- cite the CDFD Runtime separately when runtime code is used;

- do not treat any paper-local toy diagnostic as a clinical device or clinical
